# LinuxHelp

This workspace collects notes, scripts, and configuration snippets for working with Linux hosts, Raspberry Pi / Orange Pi devices, ESPHome, Git, and related setup tasks.

## Contents
- [bash.md](bash.md) — shell and command-line notes
- [git.md](git.md) — Git workflows and common commands
- [ESPHome.md](ESPHome.md) — ESPHome firmware notes
- [majordomo.md](majordomo.md) — system administration and deployment notes
- [prepareHost.ps1](prepareHost.ps1) — PowerShell helper for preparing SSH access
- [status-led.py](status-led.py) — simple Raspberry Pi status LED script

## Quick start
1. Review the relevant guide for your task.
2. Use the PowerShell helper when you need to prepare SSH access to a remote host.
3. Keep the repository organized by adding new notes as separate markdown files.

For the original wiki reference, see [Wiki!](https://github.com/SrFatCat/LinuxHelp/wiki).
